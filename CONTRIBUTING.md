# Contributing

## Docstring

1. One line summary.

Start with a verb and describe what not how.

```python
"""Mask all but the first and last character of a string"""
```

Required

2. Parameters

```python
:param s: Input string
```

Optional, if obvious

3. Return values

```python
:return: Tuple of masked string and number of replacements
```

Optional, if obvious

1. Private helpers

One line is enough, optional if obvious

1. Exceptions

Optional, if obvious

> **Note:** Write docstirng before implementing so you aren't going back 4 months later trying to understand what you wrote.
